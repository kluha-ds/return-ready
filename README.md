# AI Life Admin

Mobile-first Flutter MVP for the AI Life Admin concept.

## What it includes
- onboarding-oriented hero flow
- capture entry points for screenshot import, paper doc scan, and pasted text
- heuristic AI parsing for bills, appointments, deadlines, forms, and informational items
- review sheet with source preview, extracted fields, confidence cues, duplicate warnings, and sensitive-item handling
- save outcomes as task, reminder, calendar-event draft, or reference-only item
- home views for Inbox, Today, and Upcoming
- archive/search view for saved references and completed context
- item detail with source linkage, reminder controls, edit history, mute/snooze, and delete flow
- privacy and trust settings surface

## Run locally
```bash
flutter pub get
flutter run
```

## Verify
```bash
flutter analyze
flutter test
```
